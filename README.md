# Atelier d'Écriture Créative

Un site web interactif pour un atelier d'écriture créative, développé avec React et TypeScript.

## 🎨 Caractéristiques

- Design élégant avec une palette de couleurs naturelles (beige, vert, brun)
- Interface responsive s'adaptant à tous les appareils
- Navigation intuitive entre les différentes sections
- Pages complètes pour :
  - Accueil avec présentation des ateliers
  - À propos expliquant la mission et les valeurs
  - Blog avec articles sur l'écriture créative
  - Système de réservation pour les ateliers
  - Formulaire de contact

## 🛠️ Technologies utilisées

- React 18
- TypeScript
- React Router pour la navigation
- Styled Components pour le style
- Responsive Design

## 📋 Prérequis

- Node.js (version 14 ou supérieure)
- npm ou yarn

## 🚀 Installation

1. Clonez ce dépôt sur votre machine locale
```bash
git clone https://github.com/votre-nom/atelier-ecriture-creative.git
cd atelier-ecriture-creative
```

2. Installez les dépendances
```bash
npm install
# ou
yarn install
```

3. Lancez l'application en mode développement
```bash
npm start
# ou
yarn start
```

4. Ouvrez votre navigateur à l'adresse [http://localhost:3000](http://localhost:3000)

## 📦 Production

Pour générer une version optimisée pour la production :

```bash
npm run build
# ou
yarn build
```

Les fichiers seront générés dans le dossier `build`.
